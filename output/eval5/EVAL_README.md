# 网文复写成品 · 人工评分包(5 题材)

> 给运营/网文读者评委。**目标:用网文读者的真实口径,给 5 本成品打分**——这是校准机器评分、定位质量天花板的"真值"。
> 请**至少 2 位评委各自独立**评分(防单人偏差)。预计每本通读+打分 ~40-60 分钟。

---

## 0. 最重要的两条纪律

1. **盲评、按网文读者口径**:把它当一本普通网文来读,问自己"这书好不好看、追不追得下去",**不要因为知道是机器写的而手软或苛刻**。
2. **诚实、别给面子分**:我们要的是真值,不是好看的数字。**塌方就打塌方分**。预期这批分数大概率 45–65,不到出货线(75)很正常——这不是你打错了,是系统现状。

## 2. 怎么打分:四维 2×2 + 锚

每本给 **4 个维度** 各打 0–100 的**整体直觉分**(不是子项加总,凭读者感觉给一个数)。

| 维度 | 权重 | 看什么(子项=找理由的提示,不单独打分) |
|---|---|---|
| **故事性** | 30% | 钩子张力 / 爽点兑现 / 章末驱动 / **追读欲** |
| **笔力** | 25% | 去 AI 腔 / 对话鲜活 / 画面感 / 辨识度 |
| **人** | 25% | 角色魅力 / 代入感 / 主动性 / 成长弧 |
| **承重** | 20% | 长线结构连续 / 世界观一致 / **跨章事实自洽(人名/数值/身份/时间线有没有前后矛盾)** / 记忆点 |

**分数锚(校准刻度,务必对齐)**:

| 分段 | 含义 |
|---|---|
| **90** | 网文标准顶级好看,能让人熬夜追完、催更 |
| **75**(出货线) | 能顺畅追完,爽点/钩子在线,瑕疵不影响追读 |
| **60**(及格) | 能读,但有让人出戏的硬伤(人名串/情节矛盾/逻辑断),追读意愿一般 |
| **45**(塌方) | 频繁出戏 / 读不下去 / 想弃书 |

**承重维特别提示**:网文最容易塌在"跨章不自洽"——比如同一个人前后名字不一样、彩礼/年龄数值前后矛盾、死了又活、身份(养父/亲哥)换来换去、时间线倒退。读到这类,承重就往下压。
