> 背景:六轮工程后人工/Fable 均值卡 ~65-75(严尺 ~64),分维=故事性 72-83 稳 / 笔力 68-75 / 人 60-65(源决定)/ **承重 40-50 塌陷**。
> 第7跑实评定论(2026-06-11):承重失效全在 **prose 事实层**(死人复活/时间轴互斥/体系混用/数值倒退/人设换皮),
> plan 级账本、det 审计、局部修复 pass 都够不到——"修复抹平句面,抹不平事实"。且交付门与该缺陷类正交
> (`produce.py` 门只拦串线/过短/暗黑/跳空,维14 命中 + final_consistent=false 照常交付)。
> 重掷方差 ±5 > 单轮工程增益(第6跑铁证),继续逐类加检测器 = 打地鼠。

## B 顺序整本起草 M0(一本对照实验,不动产线)

锚源=团宠(同源已有 r3/r6/round6 三版历史分可比)。复用现有 plan,只改起草循环为**顺序**:

```
for ch in 1..60:  context = canon + 近10章全文 + 更早章摘要(growing prefix,吃 DeepSeek 前缀缓存)
```

对照组 = 同 plan 现行并行起草。三个测量:
1. **承重差分**:A2 对账环对两版各跑一遍,矛盾数对比(同时是 A2 的第二测试场);
2. **尾部质量**:第 50-60 章 vs 前 10 章的套话密度/控字达标/铁律服从(测 300k+ 前缀处退化);
3. **成本与墙钟**(缓存命中率决定成本账;预期单本墙钟 25min→40-60min,跨本并行补吞吐)。

**切换判据(三条同时成立才拍换起草层)**:顺序版矛盾数 ≤ 并行版 50%,且尾部章质量不塌,且成本 ≤2× 现状。
任一不成立 → 留在路线 A,B 降级远期。

## M0 实测结果(2026-06-11,当日落地当日测完,总成本 ~¥12.2)

### A1 交付门:✅ 唯一全胜项,已落地产线

10 本回放(docs/plans/replay_result.md):≤65 五本全拦(灵气=残缝9+final_consistent、冷战=维14、三本现行门)、≥68 三本零误拦;灰区两本被 final_consistent 拦且与坐实缺陷一致。残缝阈值定稿 8。**三判据已进 produce.py ship_issues。**

### A2 对账环:❌ 朴素形态证伪,降级 advisory

召回 9%(口径校正 18%)≪50%(docs/plans/recall_result.md)。报告几乎全落前 7 章——**1M 读得下但深处不查**,跨段长程矛盾(死 ch15 活 ch47、B超 ch24 vs 妊娠 ch45)恰好全漏。验真率 67%、¥0.45/本,当 advisory 有正价值,不进门。
**A2' 下一轮假设**:LLM 逐章抽事实表(生死/修为/身份/数值,flash 60 并发=局部抽取强项)→ 代码跨章确定性对比;与已验证 roster 模式同构,把全局推理从模型挪进代码。

### B 顺序起草:❌ 切换判据不成立,留路线 A

同 plan 两臂对照(团宠源,n=3 无金标,跳过修复 pass;output/_m0_seq/):

| 指标 | 并行(现行) | 顺序(全历史前缀) | 判定 |
|---|---|---|---|
| 章缝检出(SEAM_CHECK,主尺) | 19 | 15(-21%) | ✗ 远低于减半线 |
| POV 离群章 | 1 | 0 | 小赢 |
| 尾部质量(后10章) | 套话34/均字3688 | 套话15/均字3549/过短1 | ✓ 不塌,反而更净 |
| 成本 | ¥4.70 | ¥5.27(1.12×) | ✓ 缓存经济成立 |
| 墙钟 | 339s | 5616s(16.6×) | 预期内 |

**核心发现:起草者看见全部前文,章缝几乎没少——"起草者失明是章缝根因"被证伪。** 两臂同 plan 而缝数相近=断裂主源在 **plan 节拍跳跃**(上游),其次是注意力未必利用前文尾部(与 A2 深处不查同因)。顺序起草的真实收益是 craft 侧小幅(套话更少/POV 干净),不值 16× 墙钟。n=1 注意方差,但距减半线太远,结论稳。

### 本轮净结论与下一步

1. **承重的工程主战场回到:plan 级连续性(节拍物理交接硬化)+ A2' 事实表对账 + 门拦截 + 选优点修。** 1M 上下文两个方向(读全书/写全书)的朴素用法都不解决承重,长上下文不是银弹。
2. 分数预期修正:本轮交付分布≈"只 A"情景(交付均值 70-72,严尺),75-85 还差下一轮(A2'+选优点修+plan 缝硬化+严选源)。
3. 负结果价值:两个 M0 共花 ~¥11,避免了把起草层重写错方向(若直接切换=数周工程+16×墙钟换 -21% 章缝)。
