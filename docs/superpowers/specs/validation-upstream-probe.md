# 验证块 上游跨族探针提示词(冻结,逐字用)

不改写、不补全;只指出风险。类目限定在:境界乱序 / 修为倒退 / 性别错 / 混名认亲矛盾 / 死人复活 / 章节复制注水 / DNA身世互斥 / 人设崩 / 现代腔出戏。
**只输出一个 JSON 对象**:{"predicted":["类目",...],"why":{"类目":"依据(指 bible/plan 哪处)"}}
predicted 只填你有具体依据的类目;无风险则 predicted 为空数组。
