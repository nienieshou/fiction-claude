# 左上角 DeepSeek 余额显示 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 仪表盘左上角常驻显示 DeepSeek 余额（后端代理取数 + 前端余额行，poll 60s + 手动刷新 + 低余额告警）。

**Architecture:** 新增后端 `web/backend/balance.py`（纯归一 + async 取数，优雅降级绝不抛）+ `app.py` 路由 `GET /api/balance`；前端 `index.html` 的 `.brand` 块加余额行 + `refreshBalance()` + boot 钩子 + 60s interval。key 仅在服务器，浏览器不接触。

**Tech Stack:** FastAPI，httpx（已是依赖），python-dotenv，原生 JS（单文件前端，零构建）。

## Global Constraints

- 后端契约：`{"available": bool|null, "currency": str|null, "total": str|null, "source": "live"|"no-key"|"error", "detail"?: str}`
- `total` 是**字符串**（如 `"110.00"`）；前端阈值比较前必须 `parseFloat`
- DeepSeek 端点 `GET https://api.deepseek.com/user/balance`，`Authorization: Bearer <DEEPSEEK_API_KEY>`，`timeout=5`
- 取数取首币种 `balance_infos[0]`
- **优雅降级**：无 key→`source:"no-key"`（不调外网）；任何调用失败/超时/非2xx/结构异常→`source:"error"`；`fetch_balance` 绝不抛 → 路由不 500
- 告警分级（CNY，单本 ~¥8）：`source!=="live"`或无数→灰；`available===false`或`parseFloat(total)<10`→红；`<30`→琥珀；否则绿
- 前端 `.brand` 保留 logo+`HIKI · 产线监视台`，仅替换 `fiction-rewrite · pilot` 那行
- 刷新：boot 拉一次 + `setInterval` 60s + 余额行 `↻` 点击
- 测试零真实 API（mock httpx），不花钱

- [ ] **Step 3: 创建 `web/backend/balance.py`**

```python
"""DeepSeek 余额代理: 服务器带 key 调 /user/balance,归一后吐前端。key 不进浏览器。
优雅降级——任何失败都返回带 source 标记的 dict,绝不抛(不 500 仪表盘)。"""
from __future__ import annotations

import os

import httpx
from dotenv import load_dotenv

from . import paths

load_dotenv(paths.ROOT / ".env")          # 确保 DEEPSEEK_API_KEY 可读(与 hiki.client 一致)

_URL = "https://api.deepseek.com/user/balance"


def normalize_balance(raw: dict) -> dict:
    """DeepSeek 原始 JSON → 前端契约。取首币种 balance_infos[0];结构异常→source:error。"""
    try:
        info = raw["balance_infos"][0]
        return {"available": bool(raw.get("is_available")),
                "currency": info["currency"], "total": info["total_balance"], "source": "live"}
    except (KeyError, IndexError, TypeError):
        return {"available": None, "currency": None, "total": None,
                "source": "error", "detail": "意外的余额响应结构"}


async def fetch_balance() -> dict:
    """取余额: 无 key→no-key(不调外网); 失败/超时/非2xx→error。绝不抛。"""
    key = os.environ.get("DEEPSEEK_API_KEY")
    if not key:
        return {"available": None, "currency": None, "total": None, "source": "no-key"}
    try:
        async with httpx.AsyncClient(timeout=5) as cli:
            r = await cli.get(_URL, headers={"Authorization": f"Bearer {key}"})
        if r.status_code // 100 != 2:
            return {"available": None, "currency": None, "total": None,
                    "source": "error", "detail": f"HTTP {r.status_code}"}
        return normalize_balance(r.json())
    except Exception as e:
        return {"available": None, "currency": None, "total": None,
                "source": "error", "detail": type(e).__name__}
```

```javascript
async function refreshBalance(){
  const el=document.getElementById("balrow"); if(!el) return;
  let b; try{ b=await jget("/api/balance"); }
  catch(e){ b={source:"error",detail:e.message,total:null,available:null}; }
  if(b.source!=="live" || b.total==null){
    el.textContent="余额 — ↻"; el.style.color="var(--t4)";
    el.title="余额不可用("+(b.source||"?")+(b.detail?": "+b.detail:"")+") · 点击刷新"; return;
  }
  const sym = b.currency==="USD" ? "$" : b.currency==="CNY" ? "¥" : (b.currency||"")+" ";
  const v = parseFloat(b.total);
  let col="var(--green)";
  if(b.available===false || v<10) col="var(--red)";
  else if(v<30) col="var(--amber)";
  el.textContent="余额 "+sym+b.total+" ↻"; el.style.color=col;
  el.title="DeepSeek 余额 · 点击刷新";
}
```
