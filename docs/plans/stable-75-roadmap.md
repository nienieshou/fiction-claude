# 稳定 75 路线图(2026-06-12)· 从 R16 到可商用交付

## 0. 目标重定义(务实、可达)

**不是**"任意源每本稳定 75"(普通 A 源天花板 ~70-73,不可达)。
**是**：**「S/顶级A源 + best-of-K + 选择性交付 → 稳定交付 75±2」**;普通源降级良品线(67-72)或拒收。

依据:75 的缺口主要在**承重维(工程能攻)**,不在笔力/人(能力墙)——详见 `iteration-final-summary.md`。90 不可达(能力墙),75 是运营/工程问题(货源+方差+成本),可解。

## 1. 当前→75 的账(以暴君 S源 68 为基线)

| 来源 | 增量 | 累计 |
|---|---|---|
| 基线(S源,R15实测) | — | 68 |
| M1 换判别器(选优生效) | +1-2 | 70 |
| M2 承重补漏(消除残留硬伤) | +2-3 | 72 |
| M3 best-of-K(挑方差好的一侧) | +2-3 | 75-76 |
| M4 点修(已验证) | +2-3 | 76-78 |
| → 选择性交付门挑 ≥75 出货 | — | **稳定交付 75-77** |

账能算平。下面是按依赖排序的最小工程清单。

## 2. 最小工程清单

### M1. 选优换判别器(零 API 成本,最先做)
**改**:`slice_validate._process_scene` 的选优从 `PICK`(绝对锦标赛,实测 25%≈噪声)→ `GOLD_PK` 打分选最高(对照金标,实测 75%)。
- 依据:`discriminator-experiment.md` 实锤——产线现用 PICK 选优≈随机,没吃到模型尾部。
- **验证**:锚本(团宠/暴君)配对差分,预期 +1-2;同时扩判别器验证到 20 场景确认 75% 稳定(¥2)。
- 风险:GOLD_PK 慢(pro),全场景打分成本↑;可只对造峰场景用。

### M2. 承重补漏(消除 R16 残留硬伤)
R16 三本的承重残留硬伤(从 batch_summary):
- **死人复活**(段风凌 ch34死 ch58活):事实表生死通道已有,`repair_revivals_smart` 对"死后≥3章再现"的修复未净 → 加强:verify 确认后强制改写或降级为"假死/重伤"。
- **预告跳空**(不落天结果/调查非法医疗):结尾守卫已检出,但只拦不补 → `ENDING_FIX` 扩展为"补演预告的大事件"(而非只补收束拍)。
- **邻章章缝重述**(ch58承接ch57):macro 剧情进展唯一(R16)已降到章缝,残留靠 seam_pass 采用守卫放宽。
- **验证**:锚本承重维 58→65+,die/skip/seam 残留清零。

### M3. best-of-K 整本(买稳定性,核心)
**新增**:`batch.py` 加 `--best-of K` 模式——同源跑 K 次(不同种子),用**整本判别器**选最优交付。
- 整本判别器:对照 PK(K本 vs 标杆90锚本)或四维结构指标加权,选最高。**命门**:整本判别比场景判别难,DeepSeek 可能不可靠 → 初期用强判别器/人工选本,后期校准自动。
- K=3-5,成本 ¥7×K=¥21-35(¥50内)。
- **验证**:同源 best-of-5 的最优本 Fable 分 vs 单次均值,预期 +2-3(吃 ±5 方差的好一侧)。

### M4. 选择性交付门(75 线代理判定)
**改**:交付门从当前"零硬伤才放行"升级为"75 分代理线":
- 代理指标 = 结构指标 floor(0-LLM) ⊕ 每维 floor ⊕ 对照 PK 达标(场景胜过源+逼近90锚)。
- 达 75 代理线 → 交付;否则点修一轮;仍不达 → 拒收/降级良品线。
- **依赖偏差校准器**(HFL 训"代理分↔Fable 分"),把代理线对齐真 75。当前 HFL 37 条偏少,需扩到 100+。
- **验证**:交付门 pass↔Fable≥75 一致性(放量前置闸的解锁条件)。

### M5. 货源(运营,非代码)
- **扩源池筛 S/顶级 A**:pregrade 预筛(¥1/本),从更大源池捞 S 源(当前 40 本仅 2 本 S,产能受限)。
- **题材优先**:宫斗/虐恋(高适配)> 修仙(注水拆章,低适配)。

### M6.(可选)大 N + 微调(探 78)
- 全场景 N=12 + GOLD_PK 选优(单本 ¥40):吃模型尾部,+2-4。
- 微调(路线 K):分布上移 +3-5。仅在 M1-M5 榨干后、ROI 明确时启动。

## 3. 三个必须接受的让步(运营现实)

1. **产能受 S 源供给限制**:只产好源 → 产量 = S/顶级A源数量,不是任意万本。
2. **单本成本升到 ¥35-50**:best-of-K(×K)+ 大N + 强判别器,均价大涨(vs 当前 ¥7)。在 ¥50 上限内但产能/成本权衡变差。
3. **拒收率可能 >50%**:选择性交付 → 大量普通 A 源被拒,碰"拒收≤50%"约束。需放宽该约束,或普通源走"良品线 67-72"降级交付而非拒收。

## 4. 里程碑与杀判据

| 阶段 | 做什么 | 杀判据 |
|---|---|---|
| **P1 零成本工程** | M1 换判别器 + M2 承重补漏 | 锚本 68→72?不到 → 选优/承重路径偏弱 |

| **P4 货源+量产** | M5 筛 S 源 + 批量 | 拒收>50% 且 S 源不足 → 产能不可商用 |
