## 承重审查要点 · 剧情重复 / 抗注水

归属:**承重(长线 × 吸引力)**。剧情重复是只有跨几十章才暴露的长线病,直接削弱"撑不撑到完本"。

### 一刀判据
> **这段重复删掉后,剧情有没有损失?** 没损失 = 注水,扣分;有损失(带来新信息/新升级/新关系) = 良性套路,不扣。

良性重复(爽点节拍、升级循环、固定撒糖)是网文卖点,**不扣**。要审的只是**无增量重复 = 原地踏步**。

### 六种病态形态(对照查)
1. **回忆杀注水**:整章/大段把写过的场景用"回忆"再叙一遍凑字数。
2. **信息复述**:角色向新人物从头复述前情,展开成大篇幅。
3. **桥段克隆**:反复"获宝→突破→打脸同类反派",对手换皮、流程照搬、战力没真升级。
4. **误会循环**:同一种误会—解释—再误会反复刷,关系零推进(追妻流重灾区)。
5. **副本同质**:单元结构雷同到像复制粘贴,只换地名与 boss 名。
6. **设定反复解释**:同一金手指/世界观规则被重复科普。

### 边界(避免与其它项重复计扣)
- **vs 结构连续**:连续性管"前后矛盾/线索悬空"(断裂);抗注水管"前后过度雷同/原地循环"(冗余)。一缺一余,病灶相反。
- **vs 故事性·爽点通胀**:同一病分线记录——**短线**"又是这套"的体感疲劳记在**故事性**;**长线**结构性原地踏步、注水拉长记在**承重·抗注水**。

### 量化信号(吃增强版逐章摘要)
- **情节单元去重**:把「对手类型 × 爽点类型 × 结局」相同、且战力/格局**未抬升**的章,判为重复单元。
- **重复率 = 重复单元数 / 总单元数**;并看**分布**(均匀套路 vs 某段集中注水)。
- **回忆/复述占比**:扫摘要中"回忆/想起/说起之前"高频段,标注疑似注水区间。
- **输出形态**:结论须带**章号区间证据**,例如"第 190–260 章反复打脸三拨同质反派、战力停滞,疑似注水 70 章"。

### 复核下钻评分(仅仲裁时用,0–5)
- 5:几乎无无效重复,套路也在持续加码。
- 4:偶有复述,无伤大体。
- 3:存在 1–2 段可压缩的同质单元。
- 2:中段明显原地踏步,有成片注水。
- 1:大量回忆/复述/克隆,推进停滞。
- 0:全程靠重复凑量,主线几乎不动。

## 配套 · 增强版逐章摘要模板

为让上述长线信号(重复率、爽虐曲线、钩子密度)可**算**而非靠印象,逐章摘要在原"一句话剧情"基础上,每章追加结构化标签。

### 每章字段
`第N章 标题 | 情绪极性 | 章节功能 | 爽点类型 | 章末钩子 — 一句话剧情`

- **情绪极性**:爽 / 虐 / 平
- **章节功能**:铺垫 / 冲突 / 打脸高潮 / 日常 / 转场 / 回忆复述
- **爽点类型**:打脸 / 升级 / 获宝 / 撒糖 / 打副本 / 解谜 / 无
- **章末钩子**:有 / 无

### 示例
- **第4章 暴打刘云霆** ｜爽｜打脸高潮｜打脸｜钩子:有 — 主角一拳打飞强抢土地的无赖,直播间疯狂打赏却以为是演戏。
- **第207章 又见黑风寨** ｜爽｜冲突｜打副本｜钩子:无 — 再下同质山寨剿匪,流程同前、战力无提升(疑似重复单元)。

### 由标签可直接派生
- **爽虐曲线 / 爽点密度**:按"情绪极性 + 爽点类型"沿章号排,数平均几章一爽、有无长憋屈段。
- **钩子密度**:按"章末钩子=有"的比例与间隔。
- **重复率**:按上节"情节单元去重"口径,统计标签组合雷同且无升级的章占比。
- **节奏**:按"章节功能"分布看加速/放缓段。

> 标注成本最高性价比:只多打 4 个标签,长线三大信号即可从主观印象升级为带章号的可计算证据。
