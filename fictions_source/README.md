## 规则
- 一本一类:按内容主线归一个题材;边界本(如 种田×言情、高武×玄幻)按**主导主线**判,可再移。
- 入新书:若属未物化类,直接 `mkdir` 该规范类名再放入(沿用上表命名,勿另起别名)。
- **验证块分层无关细题材**:失败模式由跨题材特征(境界体系/大世界观/长连续性弧/大角色群)决定,风险富集 vs 量产典型 在预登记清单里单独打标,不靠本目录粒度。
